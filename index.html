Ну не писать же нам это все каждый раз! Сделаем простую функцию для регрессии:

reg_function=function(x, y){
  N=length(x)
  
  X_mean=mean(x)
  X_sd=sd(x)
  
  Y_mean=mean(y)
  Y_sd=sd(y)
  
  XY_mean=mean(x*y)

  XY_cov=sum((x-X_mean)*(y-Y_mean))/(N-1)
  r_coef=XY_cov/(X_sd*Y_sd)
  
  b=r_coef*(Y_sd/X_sd)
  a=Y_mean - b*X_mean
  if (a>0){
    sign=' + '
  } else {
    sign=''
  }
  
  res=paste0('Regression formula: ', b, 'x', sign, a)
  print(res)
  res_data=data.frame(x=x, y=y, y_calc=b*x+a)
  return(res_data)
}

R_sq=function(x, y, y_calc){
  SS_tot=sum((y-mean(y))**2)
  SS_explained=sum((y_calc-mean(y))**2)
  SS_res=sum((y-y_calc)**2)
  R2=1-(SS_res/SS_tot)
  R2_adj=1-(SS_res/((length(x)-2))/(SS_tot/(length(x)-1)))
  print(paste0('R-squared is ', R2))
  print(paste0('adj. R-squared is ', R2_adj))
  return(R2)
}

Уже куда интереснее! Что же тут есть?

  1. Оценки значений коэффициентов в колонке Estimate.

  2. t-value и p-value - коэффициенты проверяются на значимость. Значимость удобно смотреть по звездочкам справа.

  3. R-squared и adjasted R-squared. Ориентироваться лучше на второе, поскольку этот показатель надежнее

  4. F-statistics. Она является показателем того, насколько модель отличается от intercept-only model. То есть просто от оценки среднего

Выбор модели

Если мы не знаем, какие предикторы лучше включить и решаем довериться машине, то мы можем использовать stepwise алгоритм (это больше характерно для data science, чем для науки)

full_model=lm(qsec~., mtcars)
best_AIC_model_both=step(full_model, direction = "both", trace = F)
summary(best_AIC_model_both)$r.squared
## [1] 0.8642928
best_AIC_model_backward=step(full_model, direction = "backward", trace = F)
summary(best_AIC_model_backward)$r.squared
